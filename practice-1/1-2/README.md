Задание 1-2: Процессы fork/exec<br>
Требования:<br>
•Используйте: fork(), exec(), wait(), getpid(), getppid()<br>
•Обрабатывайте ошибки<br>
•Компиляция: gcc -Wall -Wextra -o task task.c<br>

Вариант 6: Создать 4 дочерних процесса. Каждый i засыпает на i секунд (1,2,3,4).<br>
После пробуждения выводит "Процесс i проснулся". Родитель ждёт всех.<br>

Используемые функции:<br>
fork() - создание нового процесса<br>
getpid() - получение идентификатора текущего процесса<br>
getppid() - получение идентификатора родительского процесса<br>
sleep() - приостановка выполнения процесса<br>
wait() - ожидание завершения дочернего процесса<br>
exit() - завершение процесса<br>

Дочерние процессы создаются почти одновременно<br>
Каждый процесс засыпает на разное время<br>
Процессы завершаются в порядке возрастания времени сна<br>
Родительский процесс получает код завершения каждого дочернего процесса<br>
Для запуска используется ./task<br>


Пример вывода: <br>
Родительский процесс начался (PID: 2529, PPID: 305)<br>
Дочерний процесс 1 создан (PID: 2530, PPID: 2529)<br>
Процесс 1 засыпает на 1 секунд...<br>
Дочерний процесс 2 создан (PID: 2531, PPID: 2529)<br>
Процесс 2 засыпает на 2 секунд...<br>
Дочерний процесс 3 создан (PID: 2532, PPID: 2529)<br>
Процесс 3 засыпает на 3 секунд...<br>

Родительский процесс (PID: 2529) ожидает завершения дочерних...<br>
Дочерний процесс 4 создан (PID: 2533, PPID: 2529)<br>
Процесс 4 засыпает на 4 секунд...<br>
Процесс 1 проснулся<br>
Дочерний процесс с PID 2530 завершился с кодом: 1<br>

Процесс 3 проснулся<br>
Дочерний процесс с PID 2532 завершился с кодом: 3<br>
Процесс 4 проснулся<br>
